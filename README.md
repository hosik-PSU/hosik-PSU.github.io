# Hosik Kim — GitHub Pages

This folder is ready to publish as a GitHub Pages site.

## Publish (quick)
1) Create a new GitHub repository (public) named `hosik-kim` (or any name you want).
2) Upload the contents of this folder to the repository root.
3) In GitHub: Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main / (root)
4) GitHub will show the public URL.

## Customize
- Edit `index.html` for content
- Edit `assets/style.css` for styling
